Этап 2. Создание главной страницы сайта


## 2.5. Изучите макет и создайте основные блоки страницы

- [+] Определите, из каких блоков состоит главная страница в макете Figma
- [+] Создайте структуру из `<header>`, `<footer>` и `<div>`
- [] Расположите блоки в `<body>` в логическом порядке (сверху вниз)

---

## 2.6. Добавьте логотип сайта в `<header>`

- [+] Скачайте изображение логотипа из макета Figma
- [+] Сохраните его в папку `assets/`
- [+] Добавьте логотип в `<header>` через `<img>`

---

## 2.7. Оформите навигационное меню сайта

> ⚠️ **Этот блок общий для всех страниц.** Создаём один раз, потом копируем.

- [+] Определите разделы меню по макету Figma (Главная, Фильмы, Сериалы и т.д.)
- [+] Создайте меню внутри `<nav>`
- [+] Внутри `<nav>` разместите заголовок и контейнер для пунктов меню
- [+] Создайте разделы меню (ссылки на страницы)
- [+] Сохраните стили навигации в `style/common.css`
- [+] Настройте отступы и расположение элементов по макету
- [ ] **Адаптивность:** настройте отображение меню для мобильной версии (бургер-меню или скрытие/перестроение)

---

## 2.8. Добавьте переключатель темной и светлой темы

- [ ] Добавьте переключатель в `<header>` (иконка из макета Figma или геометрические фигуры)
- [ ] Сохраните стили переключателя в `style/common.css` (используется на всех страницах)
- [ ] **Адаптивность:** настройте расположение переключателя в мобильной версии по макету

---

## 2.9. Оформите шапку страницы (hero-блок)

- [ ] Скачайте изображение для шапки из макета Figma
- [ ] Добавьте `<h1>` с заголовком шапки
- [ ] Добавьте `<img>` с фоновым изображением
- [ ] Настройте градиентный переход от черного цвета к изображению через CSS
- [ ] Сохраните стили в `style/index.css`

> 💡 **Как сделать градиент:** используйте `linear-gradient()` поверх `background-image` или псевдоэлемент `::after` с градиентом.

---

## 2.10. Оформите галерею с постерами фильмов

- [ ] Создайте карточки фильмов через `<div>`
- [ ] Добавьте в каждую карточку: постер (`<img>`), название, рейтинг
- [ ] Используйте изображения и названия из макета Figma (на этом этапе — заглушки)
- [ ] Добавьте стрелки переключения карточек (иконки из UI-kit Figma)
- [ ] Сохраните стили в `style/index.css`

---

## 2.11. Оформите блок со строкой поиска

- [ ] Сделайте фоновое изображение из макета
- [ ] Добавьте `<input>` для ввода текста поиска
- [ ] Добавьте кнопку `<button>Найти</button>`
- [ ] Оформите строку поиска по макету через CSS
- [ ] Сохраните стили в `style/common.css` (используется на всех страницах)

---

## 2.12. Оформите блок с основной информацией о сайте

- [ ] Сделайте фоновое изображение из макета
- [ ] Наложите затемнение через CSS (аналогично градиенту шапки)
- [ ] Добавьте `<h1>` с заголовком блока
- [ ] Добавьте `<p>` с описанием сайта (текст из макета)
- [ ] Сохраните стили в `style/index.css`

---

## 2.13. Оформите footer сайта

- [ ] Добавьте `<p>` с информацией о разработчике
- [ ] Добавьте иконки соцсетей через `<img>` (скачайте из макета)
- [ ] Настройте ссылки (`href`) или оставьте иконки неактивными
- [ ] Настройте расположение элементов через CSS
- [ ] Сохраните стили в `style/common.css` (используется на всех страницах)

---

## 2.14. Сделайте страницу адаптивной

- [ ] Используйте **Flexbox** для расположения блоков
- [ ] Добавьте **медиа-запросы** (`@media`) для разных размеров экрана
- [ ] Изучите мобильную версию в макете Figma
- [ ] Настройте корректное отображение для мобильных устройств:
  - [ ] Меню
  - [ ] Галерея
  - [ ] Карточки фильмов
  - [ ] Строка поиска
  - [ ] Footer
